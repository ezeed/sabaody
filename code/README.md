# Code Practice

Quick Node/TypeScript runner for practicing algorithms, concepts, and snippets.

## Running Files

### Run once

```bash
pnpm dev <filename>
```

### Watch mode (auto-reload on save)

```bash
pnpm watch <filename>
```

### Direct tsx usage

```bash
tsx <filename>
```

## Code Quality

### Linting

```bash
pnpm lint          # Check for issues
pnpm lint:fix      # Fix automatically
```

### Formatting

```bash
pnpm format        # Format all files
pnpm format:check  # Check formatting
```

### Type Checking

```bash
pnpm type-check    # Verify types without building
```

## Features

- ✅ Run TypeScript directly (no build step)
- ✅ Run JavaScript (`.js`, `.mjs`, `.ts`)
- ✅ Full Node.js API access
- ✅ ESM modules enabled
- ✅ Auto-reload with watch mode
- ✅ ESLint + Prettier configured
- ✅ Strict TypeScript checking
